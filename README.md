## Featured Projects

### [chat-with-pdf](https://github.com/KeshavAjagaonkar/chat-with-pdf)
RAG pipeline for conversational querying of PDF documents. Next.js frontend, FastAPI backend, pgvector for embedding storage, Gemini Embedding-001 for vectorization, Redis for caching, fully Dockerized.

### [CivicMitra](https://github.com/KeshavAjagaonkar/CivicMitra)
MERN civic grievance platform with Gemini AI integration, real-time updates via Socket.IO, geospatial indexing, a weighted-decay community priority scoring algorithm, and four-role RBAC. Hardened with rate limiting, helmet.js, and CORS lockdown.

### [DevBoard](https://github.com/KeshavAjagaonkar/DevBoard)
Distributed job application tracker I built and use daily. PostgreSQL + Prisma for data, Redis + BullMQ for the job queue, plus a companion Chrome extension for capturing postings.

### [Relic](https://github.com/KeshavAjagaonkar/Relic)
Git internals clone built from scratch using SHA-256 content-addressable storage with AWS S3 as the backing store — built to actually understand how Git works, not just to use it.

### [PulseChat](https://github.com/KeshavAjagaonkar/PulseChat)
Real-time peer-to-peer communication app built on WebRTC.
